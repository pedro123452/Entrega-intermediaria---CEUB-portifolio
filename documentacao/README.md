# Documentação

Esta seção reúne os documentos e materiais desenvolvidos durante a disciplina de Bootcamp no CEUB.

O objetivo desta pasta é centralizar os arquivos relacionados às entregas acadêmicas, apresentações, relatórios e demais conteúdos utilizados ao longo do desenvolvimento do projeto PortfolioHUB.

## Conteúdos presentes

- Entrega intermediária em PDF;
- Documentação complementar;

## Objetivo

A documentação tem como finalidade registrar de forma organizada todas as etapas do projeto, facilitando a visualização, manutenção e compartilhamento das atividades desenvolvidas durante a disciplina.

Além disso, esta organização contribui para a construção de um portfólio acadêmico e profissional mais estruturado.
