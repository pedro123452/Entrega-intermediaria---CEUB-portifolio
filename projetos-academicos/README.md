## Projetos Acadêmicos

Esta seção reúne os projetos acadêmicos desenvolvidos ao longo do semestre durante as disciplinas do curso no CEUB. Os trabalhos apresentados têm como objetivo aplicar na prática os conhecimentos adquiridos em sala de aula, envolvendo desenvolvimento de software, banco de dados, lógica de programação, versionamento e documentação de projetos.

Os projetos foram organizados de forma estruturada para facilitar a navegação, manutenção e acompanhamento da evolução das atividades realizadas durante o período acadêmico.

Entre os conteúdos desenvolvidos, destacam-se:
- Projetos em Python;
- Scripts SQL e modelagem de banco de dados;
- Exercícios práticos de programação;

Além do desenvolvimento técnico, esta organização busca construir um portfólio acadêmico e profissional, demonstrando aprendizado, evolução e aplicação prática das tecnologias estudadas ao longo do semestre.
