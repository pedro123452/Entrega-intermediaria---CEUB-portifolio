## Projetos Pessoais

Esta seção contém projetos pessoais desenvolvidos com o objetivo de praticar programação, explorar novas tecnologias e aprimorar habilidades técnicas além das atividades acadêmicas.

Os projetos apresentados representam estudos independentes, experimentos e aplicações criadas para desenvolvimento pessoal e profissional, permitindo maior aprofundamento em lógica de programação, interfaces, automações e desenvolvimento de software.

Entre os principais conteúdos desta seção, destacam-se:
- Aplicações desenvolvidas em Python;
- Automações e testes de programação;
- Estudos práticos utilizando diferentes tecnologias.

Os projetos pessoais também contribuem para a construção de um portfólio mais completo, demonstrando iniciativa, criatividade e evolução contínua no aprendizado de desenvolvimento e tecnologia.
